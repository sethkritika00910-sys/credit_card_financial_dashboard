# credit_card_financial_dashboard

## Project Overview
This project is a Credit Card Financial Dashboard made using SQL and Power BI. It helps in analyzing credit card transactions and customer data in a simple and interactive way.

## Problem Statement
The main goal of this project is to track credit card performance on a weekly basis and understand customer spending behavior, revenue trends, and default rates.

## Tools & Technologies
SQL Server – for storing and managing data
Power BI – for creating dashboards and visual reports
DAX & Power Query – for data cleaning and calculations

## Key Insights
-Analyzed total revenue and interest earned
-Tracked weekly transaction trends
-Studied customer spending based on age, income group, and card type
-Identified customer segments with higher default risk

## Methodology
-Cleaned and prepared raw CSV data
-Imported data into SQL Server
-Created interactive dashboards in Power BI
-Used DAX formulas for weekly performance analysis
-Designed charts and KPI cards for better understanding

## Features
-Weekly revenue tracking
-Customer segmentation analysis
-Transaction and default rate monitoring
-Interactive and user-friendly dashboard
